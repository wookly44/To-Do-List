## 📌 프로젝트 소개

React를 활용하여 제작한 투두리스트 웹 애플리케이션입니다. <br/>
사용자는 할 일을 등록, 수정, 삭제할 수 있으며 완료 여부에 따라 리스트를 구분하여 관리할 수 있습니다.<br/>
캐릭터와 일러스트를 활용한 UI로 사용자에게 친근한 경험을 제공하는 데 중점을 두었습니다.
(제작 시기: 2024년 상반기)

## 🖼 화면 구성

### 상단 입력 영역
- 할 일을 입력하고 추가 버튼으로 등록

### 리스트 영역
- 할 일 목록 표시
- 완료 체크 버튼
- 수정 및 삭제 버튼 제공

### 완료 리스트 버튼
- 완료된 항목만 따로 확인 가능
